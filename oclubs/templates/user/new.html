Add New Clubs


Please upload a xlsx file with a sheet named "Clubs".

Required information in xlsx(in exact order): Club Name, CL Student ID, CL Passport Name, CT ID, CT Passport Name, Club Location, Club Type

CL indicates club leader's, CT indicates club teacher's

Please input number for club type: 1 - academics, 2 - sports, 3 - arts, 4 - services, 5 - entertainment, 6 - others, 7 - school teams
